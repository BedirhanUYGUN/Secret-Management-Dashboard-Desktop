# Dokumantasyon

- `planning/` -> urun ve teknik plan belgeleri (`PLAN.md`, `ROADMAP.md`)
- `design/` -> tasarim referanslari ve stitch ciktilari
- `MVP_RELEASE_CHECKLIST.md` -> MVP release kontrol listesi
- `API_QUICKSTART.md` -> temel API kullanim kilavuzu
- `DEPLOY_RENDER_NETLIFY_SUPABASE.md` -> Render + Netlify + Supabase deployment rehberi
- `project-map.md` -> proje katman haritasi ve paylasilan dosyalar
- `module-map.md` -> modul bagimliliklari ve degisiklik yayilim matrisi
- `change-impact-checklist.md` -> degisiklik oncesi/sonrasi kontrol listesi
- `adr/` -> mimari karar kayitlari (ADR)
